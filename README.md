# vue_shop-template
这是一个基于点餐系统的电商后台管理系统，后台人员可以对用户信息进行操作（增删改查），可以查看数据以及物流信息等。
所用技术：vue + vue-router + element-ui + axios + echarts


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
